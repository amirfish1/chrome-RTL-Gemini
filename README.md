# Gemini Canvas RTL for Hebrew

A Chrome extension that automatically sets paragraph direction to right-to-left (RTL) for Hebrew text in Google Gemini's canvas editor.

## Install

Available on the [Chrome Web Store](#) or load manually:

1. Clone this repo
2. Go to `chrome://extensions`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the folder

## Privacy Policy

This extension does **not** collect, store, transmit, or share any user data. It runs entirely locally in your browser. No analytics, no tracking, no network requests. The extension only reads text content within Gemini's canvas editor to detect Hebrew characters and apply RTL styling.
